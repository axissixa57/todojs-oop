Webpack представляет популярный инструмент для сборки модулей в один файл.
webpack-dev-server, который в свою очередь будет запускать приложение.
команда "build" потребуется для компиляции всех необходимых сборок из разрозненных модулей.

$ npm start - запустится webpack для сервера, он будет следить за файлами и автоматически перезапускать его
$ npm run buid - поможет собрать сборку
$ nmp install - установка зависимостей
